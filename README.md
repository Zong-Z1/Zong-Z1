# 📰 AI News Summarizer
A lightweight news summarization tool powered by AI. Paste any article URL or raw text and get a concise, readable summary in seconds — cutting through the noise so you can stay informed without the information overload.
Built with 🤗 Hugging Face Transformers and Gradio. Perfect for keeping up with fast-moving topics without reading wall-to-wall articles.
Features:

Summarize news articles by URL or pasted text
Adjustable summary length
Clean, minimal UI via Gradio
Runs entirely in your browser — no account needed

Try it on Hugging Face Spaces: https://huggingface.co/spaces/ZongZ1/News-AI
